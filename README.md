# Spam-Message-Classifier-
A Spam Message Classifier is a machine learning model that automatically identifies whether a message is spam or ham by observing the pattern of the message and using basic NLP algorithms . This helps filter the unwanted messages to keep it safe for recipient.

## FEATURES
- Detects spam messages
- Uses Naive Bayes algorithm
- Works on text datasets 

## TECHNOLOGIES/TOOLS USED
- Python
- Machine Learning 
- Scikit-Learn
- Pandas
- Natural Language Processing (Naive Bayes)
