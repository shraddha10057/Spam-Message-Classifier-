# PROBLEM STATEMENT :
To develop an automated model of machine learning that detects the spam/ham messages by using the algorithms of NLP , by observing the format/content of the message.

# SCOPE OF THE PROJECT :
The scope of this model is to identify the spam messages easily by using ML & NLP and letting the recipient stay protected by frauds and fakes.

# TARGET USERS : The system is intended for anyone who wants to stay protected by fraud messages and quickly identify if it's spam or ham.
Like - Mobile (Non-Tech) users.
       Elderly peoples.
       Banks and Financial Institutions.

# HIGH-LEVEL FEATURES :
1. Automatic Message Classification.
2. NLP.
3. User-Friendly Interface.
4. Real time message testing.
5. High Prediction Accuracy 
6. Error Handling.
